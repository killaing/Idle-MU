MU Idle Engine V09 测试说明

启动方式：
1. 解压测试包。
2. 双击 Start-MU-Idle.cmd。
3. 程序会自动启动本地服务器，并打开浏览器进入游戏。

注意事项：
- 不建议直接双击 index.html，浏览器可能会拦截本地 JSON 数据读取。
- 启动脚本需要电脑已安装 Python，并且 python 或 py 命令可用。
- 游戏存档保存在浏览器 localStorage 中，更换浏览器或清理浏览器数据会导致本地存档消失。
- 如果 8788 端口被占用，启动脚本会自动尝试后续端口。

当前测试重点：
- 多怪物战斗显示与刷新。
- 群攻技能对多只怪物分别计算伤害。
- 怪物独立血条、蓝条、受击飘字。
- 角色头顶血条、蓝条与受击飘字。
- 自动红蓝药水设置。
- 宝石、套装、怪物贴图显示。

反馈建议：
- 请记录浏览器、操作步骤、异常截图，以及是否能稳定复现。
