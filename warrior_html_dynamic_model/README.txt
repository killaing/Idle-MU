剑士战斗动态模型 HTML Demo - 精细版

运行方式：
1. 解压压缩包。
2. 双击 index.html 即可运行。

新增功能：
- 自动战斗循环。
- 进入攻击距离后自动匹配普通攻击、重击、技能攻击。
- 技能消耗 MP，MP 自动恢复。
- 普攻、重击、技能分别有冷却与不同伤害。
- 敌人会反击，角色会进入受击动作。
- HP 归零后播放倒地死亡动作并停留在死亡帧。
- 支持复活重置和死亡动作测试。

操作：
A/D：手动移动
J：普通攻击
K：重击
L：技能攻击
R：复活
按钮：自动/手动、复活重置、测试死亡

素材说明：
当前版本基于角色设定图整理为序列帧演示，新增 hit 与 death 动作是用现有帧制作的过渡动画。正式游戏可继续替换为更高精度的透明 PNG 动作帧。
